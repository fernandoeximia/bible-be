## 🎯 Objetivo do Projeto

Criar uma aplicação web de leitura da Bíblia com foco em:
- **Consulta fácil de versículos**
- **Sistema robusto de anotações**
- **Interface limpa e intuitiva**
- **Experiência responsiva**
